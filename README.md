# Decision Tree & Random Forest from Scratch

This project focuses on implementing core machine learning algorithms from scratch without relying on high-level libraries.

## Overview
The goal was to build decision tree and random forest algorithms from first principles to understand how they work internally.

## What I did
- Implemented Decision Tree classifier using:
  - Entropy and Gini index
  - Information gain for feature selection
- Built Random Forest using:
  - Bootstrap sampling (bagging)
  - Feature randomness
- Developed full training and prediction pipeline
- Evaluated performance and compared with standard implementations
- Implemented permutation feature importance

## Dataset
- Real-world dataset with multiple numerical features and class labels

## Results
- Successfully built working implementations of both algorithms
- Achieved comparable performance to standard library implementations

## Tech Stack
- Python
- NumPy
- scikit-learn (for evaluation and comparison)

## Key Takeaways
- Gained deep understanding of tree-based models
- Learned how ensemble methods improve model robustness
- Built strong foundation in core ML algorithms
